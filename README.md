# ZARAS Premium Textiles

Luxury, responsive static website for a premium textile company in Abu Dhabi, UAE.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Run locally

```bash
npm install
npm run dev
```

## Project structure

```text
app/
components/
data/
hooks/
lib/
public/
styles/
types/
```
